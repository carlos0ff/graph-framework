# Exemplos

Scripts independentes demonstrando o uso da biblioteca. Execute cada um com:

```bash
php exemplos/<arquivo>.php
```

| Arquivo | Descrição |
|---------|-----------|
| [`01-grafo-nao-direcionado.php`](01-grafo-nao-direcionado.php) | Cria um grafo não direcionado, imprime suas adjacências (`Printer`) e executa uma BFS (`BuscaLarguraImperativa`). |
| [`02-grafo-direcionado.php`](02-grafo-direcionado.php) | Cria um grafo direcionado, mostra o grau de um vértice e executa uma BFS (`BuscaLargura`). |
| [`03-matriz-adjacencia.php`](03-matriz-adjacencia.php) | Demonstra a representação alternativa por matriz de adjacência (`AdjacencyMatrix`). |
